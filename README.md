# SafeStay

SafeStay is a responsive web application designed to help students, interns, and young professionals find trustworthy short-term housing. The platform focuses on transparency by allowing users to compare listings based on furnished availability, utilities, Wi-Fi reliability, parking, safety information, and housing details.

## Features
- Searchable housing listings
- Housing filters
- Saved/favorite listings
- Responsive mobile-friendly design
- Housing comparison cards

## Technologies
- HTML
- CSS
- JavaScript
